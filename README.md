# elabapi-python

Python library to interact with [eLabFTW](https://github.com/elabftw/elabftw) REST API v2.

## Using the helper script

~~~
# generate the library
./helper.sh generate
# build packages
./helper.sh build
# publish new version (after editing config.json, with twine installed)
./helper.sh publish
~~~
